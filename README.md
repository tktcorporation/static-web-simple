# Static Web Simple

A simple example to render html with docker.

## Get Started

### Install Docker

https://docs.docker.com/get-docker/

### Start A Service

```bash
$ docker-compose up
```

### Open The Page

Open `http://localhost:8000/public/index.html`

Access docker host from a web browser.  
(It's usually like `0.0.0.0`, `localhost`)
